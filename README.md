# 📄 Portafolio & Hoja de Vida - Esteban Manuel López Rivero (v2026)

Hoja de Vida web interactiva y moderna con diseño glassmorphism oscuro, carrusel 3D de proyectos, chatbot inteligente y PDF descargable con foto. Refleja la trayectoria técnica, comercial y empresarial del autor.

**🌐 Web:** [hoja-de-vida-v1.vercel.app](https://hoja-de-vida-v1.vercel.app/)

---

## 🚀 Características Principales

- **Diseño Glassmorphism Oscuro** — Navbar fija con efecto scroll, hero con gradiente animado, tarjetas con blur y bordes translúcidos.
- **Carrusel 3D Interactivo** — 12 proyectos web con auto-rotación (4s), controles táctiles, hover pause y navegación por dots.
- **Chatbot Inteligente** — Asistente virtual integrado que responde sobre experiencia laboral, educación técnica/comercial, proyectos, certificaciones y habilidades.
- **🎯 Palabras clave del chat:** `proyectos`, `experiencia`, `trading`, `gaming`, `servidores`, `redes`, `soporte`, `estudios`, `certificados`, `habilidades`, `contacto`, `pdf`, `sena`, `contabilidad`, `finanzas`, `tienda`, `altos del noval`, `inventario`, `certificado_tienda`, `movivip`, `detener`.
- **PDF Profesional** — Generado con `fpdf2` + Arial TrueType (Unicode), incluye foto (`FOTO 3X4.jpg`), experiencia técnica y comercial, certificaciones SENA. Peso: 260 KB.
- **Certificaciones Interactivas** — Galería con 14 certificados (CENSA, SENA, Platzi) en modal con zoom y descarga.
- **Línea de Tiempo Profesional** — 5 experiencias ordenadas: comercial (Tienda Altos del Noval), técnica (Cabarcas Sarmiento), empresarial (MoviVIP Network), desarrollo web y creación de contenido.
- **Contacto Directo** — WhatsApp con mensaje predeterminado, LinkedIn, email y ubicación.
